gimp_cairo_set_source_rgb ()

void                gimp_cairo_set_source_rgb           (cairo_t *cr,
                                                         const GimpRGB *color);

Sets the source pattern within cr to the solid opaque color described by color.

This function calls cairo_set_source_rgb() for you.

 Cairo context

color :
