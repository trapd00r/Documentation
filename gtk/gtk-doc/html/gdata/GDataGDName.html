Description

GDataGDName represents a "name" element from the GData specification.

Given a name such as Sir Winston Leonard Spencer-Churchill, KG, the properties of the GDataGDName should be set as follows:

"given-name"

Winston

"additional-name"

Leonard

"family-name"

Spencer-Churchill

"prefix"

Sir

"suffix"

KG
