Using GTK+ on DirectFB

The DirectFB port of GTK+ is an implementation of GDK (and therefore GTK+) on top of the DirectFB libraries.

Like the obsolete linux-fb port of GTK+, it runs on the Linux framebuffer, but the DirectFB libraries provide additional features like hardware graphics acceleration, input device handling and abstraction and an integrated windowing system.

Build requirements

Beyond the usual GTK+ build requirements, the DirectFB backend (obviously) needs the DirectFB libraries (at least 0.9.21) and cairo compiled with DirectFB support.
