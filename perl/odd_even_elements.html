Filter out every second value of a list

Filter out every second value of a list, starting with the first

my $t = 1;
grep { $t^=1 } LIST

[download]
my $t = 1;
grep { $t = 1-$t } LIST

[download]
my $t = 1;
grep { ++$t % 2 } LIST

[download]
map { $_%2 ? $ARRAY[$_] : () } 0..$#ARRAY

[download]

>perl -le"$t=1; print grep { $t^=1 } qw( a b c d e );"

Transform a list based on odd/even-ness of index

my $t = 1;
map { ($t^=1) ? EXPR_FOR_ODD : EXPR_FOR_EVEN } LIST

[download]
my $t = 1;
map { ($t = 1-$t) ? EXPR_FOR_ODD : EXPR_FOR_EVEN } LIST

[download]
my $t = 1;
map { (++$t % 2) ? EXPR_FOR_ODD : EXPR_FOR_EVEN } LIST

[download]
map { $_%2 ? EXPR_FOR_ODD : EXPR_FOR_EVEN } 0..$#ARRAY

[download]

Example: 

>perl -le"$t=1; print map { ($t^=1) ? uc : lc } qw( a b c d e );"
aBcDe

[download]
>perl -le"$t=1; print map { ($t=1-$t) ? uc : lc } qw( a b c d e );"
aBcDe

[download]
>perl -le"$t=1; print map { (++$t % 2) ? uc : lc } qw( a b c d e );"
aBcDe

[download]
>perl -le"@a = qw( a b c d e ); print map { $_%2 ? uc($a[$_]) : lc($a[
+$_]) } 0..$#a;"
aBcDe

[download]

Usage note

None of the above are particularly self-explanatory, so a comment is in order if they their use if not obvious from the context.

    One of my favorite subs:

    sub map_pairs(&@) {
  my $f = shift;
  my @res;

  no strict 'refs';
  no warnings 'once';
  my $caller = caller;
  local(*{$caller."::a"}) = \my $a;
  local(*{$caller."::b"}) = \my $b;

  push @res, $f->($a,$b) while ($a, $b) = splice @_, 0, 2;
  return @res;
}

    [download]

    gives:

    map_pairs { $a } qw/a b c d/; # ac
map_pairs { $b } qw/a b c d/; # bd
map_pairs { uc($a),lc($b) } qw/a b c d/; # AbCd

        It will be a problem if you are running under strict. However, if the caller sets up $c as a package variable it will work.

        sub map_triples(&@) {
  my $f = shift;
  my @res;

  no strict 'refs';
  my $caller = caller;
  local(*{$caller."::a"}) = \my $a;
  local(*{$caller."::b"}) = \my $b;
  local(*{$caller."::c"}) = \my $c;

  push @res, $f->($a,$b,$c) while ($a, $b, $c) = splice @_, 0, 3;
  return @res;
}

use 5.010;
use warnings;
use strict;
our $c;

say for map_triples { $a + $b + $c } 1..12;
