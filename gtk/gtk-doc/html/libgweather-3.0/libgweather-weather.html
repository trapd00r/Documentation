GSList *            weather_info_get_forecast_list      (WeatherInfo *info);

Returns :

 list of WeatherInfo* objects for the forecast. The list is owned by the 'info' object thus is alive as long as the 'info'. This list is filled only when requested with type FORECAST_LIST and if available for given location. The 'update' property is the date/time when the forecast info is used for.
