Clutter-Gtk Overview

Clutter-Gtk is an integration library for the Clutter toolkit providing access and integration for the GTK+ library.

Clutter-Gtk allows embedding a Clutter scene graph into a GTK+ application; retrieving themeing information; integrate GTK+ data structures and object classes with Clutter actors.
