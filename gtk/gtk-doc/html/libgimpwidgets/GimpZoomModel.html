GimpZoomModel

typedef struct _GimpZoomModel GimpZoomModel;

enum GimpZoomType

typedef enum {
  GIMP_ZOOM_IN,        /*< desc="Zoom in"  >*/
  GIMP_ZOOM_OUT,       /*< desc="Zoom out" >*/
  GIMP_ZOOM_IN_MORE,   /*< skip >*/
  GIMP_ZOOM_OUT_MORE,  /*< skip >*/
  GIMP_ZOOM_IN_MAX,    /*< skip >*/
  GIMP_ZOOM_OUT_MAX,   /*< skip >*/
  GIMP_ZOOM_TO         /*< skip >*/
} GimpZoomType;

the zoom types for GimpZoomModel.

GIMP_ZOOM_IN

 zoom in

GIMP_ZOOM_OUT

 zoom out

GIMP_ZOOM_IN_MORE

 zoom in a lot

GIMP_ZOOM_OUT_MORE

 zoom out a lot

GIMP_ZOOM_IN_MAX

 zoom in as far as possible

GIMP_ZOOM_OUT_MAX

 zoom out as far as possible

GIMP_ZOOM_TO

 zoom to a specific zoom factor
