gimp_cairo_set_focus_line_pattern ()

gboolean            gimp_cairo_set_focus_line_pattern   (cairo_t *cr,
                                                         GtkWidget *widget);

Sets color and dash pattern for stroking a focus line on the given cr. The line pattern is taken from widget.

 Cairo context

widget :

Returns :
